# Project Summary
The Superb Crypto Bot is an automated trading solution that operates on cryptocurrency exchanges, designed to enhance trading efficiency through advanced strategies. It utilizes techniques such as Fibonacci retracement levels, RSI, and EMA indicators to make data-driven trading decisions. The bot's primary goal is to optimize account management, execute trades, and implement robust risk management protocols, catering to both novice and experienced traders.

# Project Module Description
The project consists of several key modules:
- **Bot Controller**: Manages trading logic, including execution and account management.
- **Accounts Service**: Facilitates account addition, validation, and management.
- **Trading Strategy**: Implements core trading strategies, including entry and exit logic based on market conditions.
- **Responsive Web Interface**: Provides a mobile-responsive dashboard for user interaction.

# File Description Inventory
- **bot_fib_scoring.py**: Contains the core logic for trading strategies, including the BotController class that manages trading operations and methods for placing orders and validating accounts.
- **accounts_service.py**: Provides functions for managing accounts, including adding, deleting, and validating accounts.
- **templates/**: Contains HTML templates for the responsive web interface, including base layout, dashboard, accounts management, and configuration settings.
- **BUG_FIXES.md**: A comprehensive document detailing critical bugs found in the bot and their resolutions.
- **config.json**: Configuration file for setting parameters such as allowed coins, trade limits, and risk management rules.

# Technology Stack
- **Python**: Programming language for the bot's implementation.
- **PyBit**: Library used for interacting with the Bybit trading API.
- **JSON**: Format for configuration and data storage.
- **HTML/CSS**: Used for the responsive web interface, enhanced with Tailwind CSS for styling.

# Usage
To set up and run the Superb Crypto Bot, follow these steps:
1. **Install Dependencies**: Ensure you have Python installed, then install required packages:
   ```bash
   pip install pybit
   ```
2. **Configure Settings**: Modify `config.json` to set your preferred parameters such as allowed coins and risk settings.
3. **Run the Bot**: Execute the bot using:
   ```bash
   python app/bot_fib_scoring.py
   ```
4. **Access the Web Interface**: Use the responsive web interface to monitor and manage trading activities.
